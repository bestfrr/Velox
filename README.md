# Velox Messenger

## Project Documentation

### Overview
Velox Messenger is a fast, reliable messaging application designed to provide users with real-time communication capabilities.

### Features
- Instant Messaging
- Group Chats
- Multimedia Sharing
- End-to-End Encryption

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bestfrr/Velox.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Velox
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Usage
To start the application, run:
```bash
npm start
```
